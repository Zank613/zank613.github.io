CenterScript Documentation

CenterScript (.src) is the standard automation language for CenterOS. It allows users to write scripts that interact with system APIs, network interfaces, and file storage.

Scripts must be compiled using the csc command in the Terminal before execution.

1. Syntax Basics

Variables:
var x = 10;
var name = "User";

Logic:
if (x > 5) { print("Big"); } else { print("Small"); }

Loops:
while (x < 10) { x = x + 1; }

2. System API

Console.print(msg) - Output text to the terminal.
Console.input(prompt) - Pause execution and get user input.
App.launch(appId) - Open an application (e.g. "browser", "miner").
Sys.getHeat() - Returns current police heat level (0-100).
Sys.time() - Returns current time in minutes.

3. Network API

Net.scan() - Returns a comma-separated string of visible SSIDs.
Net.connect(ssid) - Attempts to connect to a WiFi network.

4. File API

File.read(filename) - Returns content of a file from Documents/Desktop.
File.write(filename, content) - Creates or overwrites a file.
File.exists(filename) - Returns 1 (true) or 0 (false).

5. Math & Helper API

Math.floor(x), Math.round(x), Math.random()
String.length(str)

6. Examples

Auto-Miner Script:

var heat = Sys.getHeat();
if (heat < 20) {
print("Safe to mine.");
App.launch("miner");
} else {
print("Too hot!");
}
